LaTeX Size Commands

To find document dimensions in LaTeX:

\texttt{\string\textwidth}: \the\textwidth
\texttt{\string\textheight}: \the\textheight
\texttt{\string\paperwidth}: \the\paperwidth
\texttt{\string\paperheight}: \the\paperheight
